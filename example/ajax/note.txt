คำอธิบาย:

ไฟล์ fetch_data.php:

ใช้ PDO ในการเชื่อมต่อกับฐานข้อมูล MySQL อย่างปลอดภัยและมีประสิทธิภาพ
เตรียมและ execute คำสั่ง SQL เพื่อดึงข้อมูลจากตาราง users
ดึงข้อมูลทั้งหมดและแปลงเป็นรูปแบบ JSON เพื่อส่งกลับไปยัง client (browser)
ไฟล์ index.html:

ใช้ jQuery AJAX เพื่อส่งคำขอแบบ GET ไปยังไฟล์ fetch_data.php
กำหนดให้รับข้อมูลในรูปแบบ JSON
เมื่อได้รับข้อมูลสำเร็จ จะนำข้อมูลมาสร้างตาราง HTML และแสดงผลใน <div> ที่มี id="userTable"
มีการจัดการกรณีที่ไม่มีข้อมูล หรือเกิดข้อผิดพลาดในการดึงข้อมูล
วิธีการใช้งาน:

สร้างฐานข้อมูล MySQL ชื่อ your_database (หรือชื่อที่คุณต้องการใช้)
สร้างตารางชื่อ users ในฐานข้อมูลนั้น โดยมีคอลัมน์ id (INT, AUTO_INCREMENT, PRIMARY KEY) และ name (VARCHAR(255), NOT NULL)
เพิ่มข้อมูลตัวอย่างลงในตาราง users
แก้ไขข้อมูลการเชื่อมต่อในไฟล์ fetch_data.php ($host, $dbname, $username, $password) ให้ตรงกับข้อมูลของฐานข้อมูลของคุณ
บันทึกไฟล์ fetch_data.php และ index.html ไว้ใน directory เดียวกันบน web server ของคุณ
เปิดไฟล์ index.html ผ่าน web browser ของคุณ
คุณควรจะเห็นข้อมูลจากตาราง users แสดงในรูปแบบตารางบนหน้าเว็บของคุณครับ
