📌 แหล่งที่มาหลัก ๆ
1. Unicode Standard
แต่ละไอคอนมีรหัส (code point) เช่น
✅ = U+2705 (White Heavy Check Mark)
⚠️ = U+26A0 (Warning Sign)
💡 = U+1F4A1 (Light Bulb)
สามารถดูได้ที่เว็บ Unicode.org

2. Emoji Keyboard / Input
Windows → Win + . (ปุ่มวินโดวส์ + จุด)
macOS → Control + Command + Space
มือถือ → ใช้คีย์บอร์ด emoji ได้เลย

3. Copy & Paste
เว็บไซต์เช่น Emojipedia

💡 การใช้งาน
สามารถใส่ใน HTML/JS/ข้อความได้ตรง ๆ
หรือใช้รหัส Unicode escape เช่นใน JavaScript

console.log("\u2705 Success!"); // ✅ Success!
console.log("\u26A0 Warning!"); // ⚠️ Warning!
